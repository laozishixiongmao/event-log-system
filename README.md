# Event Log System（事件日志系统）
## 项目简介
这是一个轻量级全栈事件日志系统，支持事件创建、列表展示、按类型筛选功能，15 分钟内完成，体现全栈前后端联动能力。

## 技术栈
- 后端：Node.js + Express（接口设计、内存存储、跨域处理）
- 前端：HTML + 原生 JavaScript（Fetch API 调用、DOM 渲染、表单交互）
- 其他：cors 中间件（解决跨域问题）

## 快速运行步骤（审阅者一键启动）
1.  确保本地已安装 Node.js（LTS 版本）
2.  进入项目根文件夹，打开终端执行：`npm install`（安装依赖）
3.  终端继续执行：`npm start`（启动后端服务）
4.  双击 `index.html` 文件，用浏览器打开即可使用

## 核心功能
1.  事件创建：通过下拉框选择事件类型（info/error/warning），输入描述后提交
2.  列表展示：页面加载自动渲染所有事件，表格清晰展示 ID、类型、描述、创建时间
3.  类型筛选：支持按事件类型筛选，切换「all」可查看全部事件
4.  数据验证：后端验证提交参数非空，空参数返回 400 错误
5.  自动刷新：表单提交成功后，自动刷新事件列表，无需手动操作
